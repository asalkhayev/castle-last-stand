# 🏰 Castle Last Stand

A dark medieval survival game built for PRESS PLAY 2026.

Fight waves of enemies, choose your battlefield, and survive as long as possible.

---

## 🎮 Gameplay

- Move: WASD or Arrow Keys
- Attack: Mouse Click or Space
- Dash: Shift
- Survive waves of enemies
- Choose level and difficulty before starting

---

## 🧭 Game Flow

1. Main Menu  
2. How To Play  
3. Level Selection  
4. Gameplay  
5. Game Over / Restart  

---

## 🗺️ Levels

- Castle — balanced gameplay  
- Forest — faster enemies  
- Crypt — harder survival  

---

## ⚙️ Features

- Custom UI with image-based buttons  
- Multiple enemy types  
- Particle effects  
- Difficulty scaling  
- Scene-based menu system
